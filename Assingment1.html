HtMl Elements :

This section includes implementation of attributes, elements, lists, paragraph, headings HTML elements are the combination of start tag, closing tag and attributes
Here are some examples of Html Elements :
  1. < a >content< / a >
  2. < p >content< / p >
  3. < image >content< / image >
  4. < form >content< / form >
  5. < li >content< / li >

Attributes :

Html Attributes provides special propeties and information to the html tag. Attributes are always written within start tag.

Syntax of Attribute :

< start_tag attribute_name = "value_of_attribute" > content < / closing_tag >

Example of some Attributes :


  • id = id_of_tag : Global Attribute
  • class = class_of_tag : Globale attribute
  • type = 1 or A : Local attribute of list
  • src = ./directory : Local attribute of image, audio, video tag
  • target = _blank : local attribute of form tag
